# 🍕 Food Classifier ML Service

Микросервис для классификации изображений еды с помощью машинного обучения.

## 🚀 Что делает

Загружаешь фото блюда — получаешь топ-5 предсказаний с вероятностями. Модель обучена на ImageNet и распознает более 1000 классов, включая десятки видов еды (пицца, бургер, суши, салат и т.д.).

## 🛠 Стек

- **Python 3.9+**
- **FastAPI** — REST API
- **PyTorch + Torchvision** — MobileNetV3 (Transfer Learning)
- **Gradio** — интерактивный UI
- **Pillow** — обработка изображений
