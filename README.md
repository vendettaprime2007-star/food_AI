# 🍕 Food Classifier AI Service

## 🚀 Что делает

Загружаешь фото блюда — получаешь топ-5 предсказаний с вероятностями. Модель обучена на ImageNet и распознает более 1000 классов, включая десятки видов еды (пицца, бургер, суши, салат и т.д.).
